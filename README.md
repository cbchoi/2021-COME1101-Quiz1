# 2021-COME1101-Quiz1
## Quiz1은 주어진 폴더에 있는 그림 파일을 확인하고 quiz1.cpp 파일을 생성하여 그림 파일의 코드를 기입하여 실행하고 이를 github에 올리는 것을 확인합니다. 

1. 사용 명령어
 - cd: 디렉토리 변경
   사용예: cd 20210916
 - clang++: 컴파일러, 
   사용예: clang++ -o quiz1 quiz1.cpp

2. 감점기준
 - 컴파일이 되지 않음: 0점
 - quiz1.cpp 파일이 없음: 0점
 - 컴파일이 되나 다른 결과를 생성: 0.5점
 - 컴파일이 되고 실행되나 런타임 에러 발생: 0점
